## ¿Qué es Diga?

DIGA es una plataforma no-code diseñada para ser **simple, potente y accesible**. Ya sea que estés construyendo agentes para atención al cliente, gestión de citas o cualquier otro caso de uso que requiera voz, tenemos todo lo que necesitas.

DIGA ofrece una solución completa para crear agentes conversacionales que atienden llamadas de forma natural. La plataforma permite gestionar múltiples proyectos con suscripciones independientes, conectar con sistemas externos mediante MCP o APIs personalizadas, y obtener números de teléfono directamente desde la interfaz o conectar los existentes vía SIP trunk.

Ve y construye algo extraordinario.
