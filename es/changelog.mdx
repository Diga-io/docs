<Update label="20 de junio de 2026" description="v1.1.17" tags={["Nuevas features"]}>

## Flujos pre-llamada en salientes e interpolación de variables más completa

- **Flujos pre-llamada para llamadas salientes**: los flujos pre-llamada ahora se ejecutan antes de marcar el número en las llamadas salientes, de modo que el agente puede preparar el contexto igual que en las llamadas entrantes.
- **Variables dinámicas en los nodos de camino**: las variables dinámicas ahora se interpolan dentro de los prompts de los nodos de los caminos conversacionales y en las condiciones de las ramas, lo que te permite personalizar los prompts y el enrutamiento por llamada.
- **Variables inyectadas en los logs de llamada a función**: las variables inyectadas en una llamada a función durante la llamada ahora se muestran en los logs de llamada a función, lo que facilita depurar qué pasó el agente a cada herramienta.

</Update>

## Trazas de llamadas con vista en cascada

- **Trazas distribuidas para cada llamada**: cada llamada ahora genera una traza que desglosa la conversación en spans individuales — turnos de conversación, generación de la IA, síntesis de voz, ejecuciones de herramientas y transiciones de camino.
- **Interfaz en cascada**: visualiza la línea de tiempo completa de la llamada en una vista en cascada para ver exactamente cuánto tardó cada paso y cómo se relacionan entre sí.
- **Diagnostica el rendimiento**: identifica rápidamente los pasos lentos, verifica los flujos conversacionales y revisa las ejecuciones de herramientas con sus argumentos y tiempos.

</Update>

## Flujos pre-llamada y variables impulsadas por flujos

### Flujos pre-llamada

- **Prepara cada llamada antes de que empiece**: el nuevo trigger **Pre-llamada** ejecuta un flujo en el momento en que empieza una llamada, antes de que el agente salude. El agente espera el resultado y lo usa para personalizar la conversación.
- **Acción Devolver datos de llamada**: envía un texto adicional para el prompt y variables dinámicas de vuelta al agente, para que empiece la llamada conociendo ya el nombre, la cuenta o el contexto del contacto.
- **Controla las llamadas**: opcionalmente aborta una llamada si el flujo falla o agota el tiempo de espera — útil para listas de permitidos, comprobaciones de elegibilidad u horarios de atención. El tiempo de espera es configurable por flujo (hasta 55 segundos).
- **Asígnalos donde quieras**: añade flujos pre-llamada a agentes de prompt único en la sección Flujos, o a caminos conversacionales mediante el nuevo nodo **Flujos pre-llamada** encima del nodo de inicio.

### Variables dinámicas impulsadas por flujos

- **Variables proporcionadas por flujos**: las variables que produce un flujo pre-llamada ahora aparecen en el panel de variables dinámicas del agente como solo lectura, con la etiqueta "Proporcionada por [flujo]", y en el autocompletado de `{{`.
- **Parámetros dinámicos para flujos durante llamada**: un parámetro de Llamada a función ahora puede tomar su valor de una variable dinámica en lugar de ser extraído por el agente — configura su **Origen del valor** como **Variable dinámica**.
- **Variables en los caminos**: las variables dinámicas ahora se interpolan en los prompts de los nodos de conversación y en las condiciones de las ramas.

### Categorías de flujos

- **Nueva categoría "Lanzador de llamadas"**: los flujos que inician llamadas salientes con la acción Hacer llamada ahora se etiquetan como **Lanzador de llamadas**, distintos de los nuevos flujos **Pre-llamada**.

</Update>

## Logs, RAG y Workflows

### Logs de llamadas

- **Registro detallado de eventos**: Cada llamada genera logs estructurados que cubren todo su ciclo de vida — desde la conexión hasta la finalización
- **10 categorías de eventos**: Ciclo de vida de la llamada, mensajes del usuario, respuestas del agente, herramientas de llamada, integraciones, caminos conversacionales, transferencias y webhooks
- **Logs externos para clientes**: Sistema de logs dual que filtra automáticamente la información técnica interna y expone solo los eventos relevantes para el usuario a través de la API
- **Trazabilidad completa**: Cada evento incluye timestamps, IDs de traza, y datos contextuales para depurar y analizar el comportamiento del agente

### RAG (Retrieval-Augmented Generation)

- **Base de conocimiento para agentes**: Sube documentos y URLs para que tus agentes accedan a información relevante durante las conversaciones
- **Búsqueda semántica**: El sistema indexa y busca automáticamente en tu contenido para dar respuestas precisas y contextuales
- **Múltiples fuentes**: Ingesta archivos, URLs y texto, con descubrimiento automático de páginas desde un dominio
- **Gestión de conocimiento**: Añade, actualiza y elimina elementos de la base de conocimiento de tus agentes

### Workflows

- **Automatización visual**: Crea flujos de trabajo sin código que se ejecutan durante o después de las llamadas
- **Workflows post-llamada**: Automatiza tareas al finalizar una llamada, como actualizar un CRM, enviar emails o registrar datos
- **Workflows durante la llamada**: Permite al agente ejecutar herramientas en tiempo real, como agendar citas o consultar datos
- **Ejecución síncrona y asíncrona**: Elige si el agente espera la respuesta del workflow o continúa la conversación
- **Plantillas predefinidas**: Usa plantillas listas para configurar workflows comunes rápidamente

</Update>

## Lanzamiento inicial de Diga


### Construcción de agentes

- **Prompts únicos**: Crea agentes conversacionales mediante prompts simples y directos
- **Caminos conversacionales**: Define flujos de conversación complejos con nodos y rutas personalizadas
  - Soporte para múltiples tipos de nodos (mensaje, pregunta, decisión, acción)
  - Editor visual para diseñar el flujo conversacional
- **Personalización de llamadas**: Configura el comportamiento y características de las llamadas
  - Voces personalizables
  - Configuración de idioma y tono
  - Parámetros de velocidad y volumen
- **Control de versiones**: Gestiona diferentes versiones de tus agentes de forma segura
  - Historial completo de cambios
  - Rollback a versiones anteriores
  - Comparación entre versiones
- **Herramientas de llamada**: Añade capacidades especiales durante las conversaciones
  - Transferencia de llamadas
  - Envío de mensajes
  - Ejecución de acciones personalizadas

### Integraciones

- **Integraciones HTTP**: Conecta tus agentes con APIs externas
  - Webhooks entrantes y salientes
  - Autenticación flexible (API Key, Bearer Token, OAuth)
  - Mapeo de datos personalizado
- **Integraciones MCP**: Soporte para Model Context Protocol
  - Integración nativa con herramientas MCP
  - Extensibilidad para casos de uso avanzados
- **Añadir integraciones a agentes**: Sistema modular para conectar múltiples integraciones

### Telefonía

- **Compra de números telefónicos**: Adquiere números directamente desde la plataforma
  - Disponibilidad en múltiples países
  - Búsqueda por código de área
  - Activación inmediata
- **SIP Trunks**: Configura y gestiona tus propios trunks SIP
  - Llamadas entrantes y salientes
  - Configuración de enrutamiento
  - Soporte para múltiples proveedores

### Gestión de plataforma

- **Proyectos**: Organiza tus agentes y recursos
  - Espacios de trabajo aislados
- **Facturación**: Control detallado de costos
  - Planes flexibles
  - Monitoreo de consumo en tiempo real
  - Reportes de uso
  - Límites configurables

### Analíticas y monitoreo

- **Monitoreo**: Visibilidad completa del comportamiento de tus agentes
  - Transcripciones completas

### Documentación y herramientas

- **API Reference completa**: Documentación interactiva con OpenAPI
  - Ejemplos en múltiples lenguajes (Python, JavaScript, cURL, Go)
  - Playground interactivo
  - Autenticación y ejemplos de código
- **Documentación en español**: Guías completas y tutoriales
- **Documentación en inglés**: Versión en inglés disponible
