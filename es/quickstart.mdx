## Comienza en tres pasos

Ponte en marcha con Diga rápidamente.

### Paso 1: Configura tu cuenta

<AccordionGroup>
  <Accordion icon="user-plus" title="Crea tu cuenta">
    Regístrate para obtener una cuenta de Diga y comenzar con nuestra plataforma de IA de voz.
    
    Visita nuestro [panel de control](https://dashboard.diga.io) para crear tu cuenta.
  </Accordion>
  <Accordion icon="key" title="Obtén tu clave API">
    Una vez que hayas iniciado sesión, navega a tu panel de control para obtener tu clave API.
    
    Necesitarás esta clave para autenticar tus solicitudes API.
    
    <Tip>Mantén tu clave API segura y nunca la compartas públicamente.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 2: Realiza tu primera llamada API

<AccordionGroup>
  <Accordion icon="phone" title="Crea tu primera llamada">
    Usa nuestra API para crear tu primera llamada de voz.

    ```bash
    curl -X POST https://api.diga.io/v1/calls \
      -H "Authorization: Bearer TU_CLAVE_API" \
      -H "Content-Type: application/json" \
      -d '{
        "agent_id": "tu_agent_id",
        "phone_number": "+1234567890"
      }'
    ```

    <Tip>Consulta nuestra [Referencia API](/api-reference/call/create_call) para más detalles.</Tip>
  </Accordion>
</AccordionGroup>

### Paso 3: Explora más funciones

<Accordion icon="rocket" title="Descubre las capacidades de Diga">
  1. Crea agentes personalizados con diferentes voces y comportamientos
  2. Configura bases de conocimiento para tus agentes
  3. Configura webhooks para notificaciones en tiempo real
  4. Monitorea tus llamadas y analiza las transcripciones

  Visita nuestra [Referencia API](/api-reference/call/get_call) para explorar todas las funciones disponibles.
</Accordion>
