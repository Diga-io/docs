<Info>
  **Requisitos previos**:
  - Node.js versión 19 o superior
  - Un repositorio de documentación con un archivo `docs.json`
</Info>

Sigue estos pasos para instalar y ejecutar Mintlify en tu sistema operativo.

<Steps>
<Step title="Instala el CLI de Mintlify">

```bash
npm i -g mint
```
</Step>

<Step title="Previsualiza localmente">

Navega a tu directorio de documentación donde se encuentra tu archivo `docs.json`, y ejecuta el siguiente comando:

```bash
mint dev
```

Una vista previa local de tu documentación estará disponible en `http://localhost:3000`.

</Step>
</Steps>

## Puertos personalizados

Por defecto, Mintlify usa el puerto 3000. Puedes personalizar el puerto en el que se ejecuta Mintlify usando la bandera `--port`. Por ejemplo, para ejecutar Mintlify en el puerto 3333, usa este comando:

```bash
mint dev --port 3333
```

Si intentas ejecutar Mintlify en un puerto que ya está en uso, usará el siguiente puerto disponible:

```md
El puerto 3000 ya está en uso. Intentando 3001 en su lugar.
```

## Solución de problemas

Aquí te mostramos cómo resolver algunos problemas comunes al trabajar con el CLI.

<AccordionGroup>
  <Accordion title="Mintlify no se está cargando">
    Actualiza a Node v19 o superior. Ejecuta `mint install` para solucionar problemas adicionales.
  </Accordion>
  <Accordion title="Puerto ya en uso">
    Mintlify usará automáticamente el siguiente puerto disponible si el puerto predeterminado está en uso.
  </Accordion>
</AccordionGroup>
